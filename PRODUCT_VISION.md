# Product Vision

## Mission

To build the most practical football performance analysis platform for analysts working in real football environments.

## Target Users

- Performance Analysts
- Coaches
- Assistant Coaches
- Academy Analysts
- Semi-professional clubs
- Non-league clubs

## Core Principles

### Football First

The software should answer football questions rather than simply display statistics.

### Simplicity

If a feature doesn't save analysts time, it doesn't belong.

### Video + Data

Every statistic should link back to the relevant video clip.

### Analyst Driven

Every feature is developed from real analysis work rather than assumptions.

## Long-Term Vision

Barghouth Footy Analytics becomes the daily workspace for football analysts, replacing spreadsheets, manual reports and disconnected tools.
