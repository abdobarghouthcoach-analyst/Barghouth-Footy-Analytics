# Barghouth Footy Analytics

A modern football performance analysis platform built by an analyst, for analysts.

## Vision

Barghouth Footy Analytics automates repetitive analysis tasks, connects football data with video, and helps coaches make better decisions faster.

The platform is being developed alongside real club analysis, ensuring every feature solves an actual problem faced by performance analysts.

## Current Status

🚧 Early Development (Milestone 0)

## Planned Features

- Match Management
- Video Upload
- Event Database
- xG Engine
- Tactical Dashboard
- Player Reports
- Opposition Reports
- PowerPoint & PDF Reports
- AI-assisted Video Analysis
